# 二极管分类、物理机制与应用场景参考

> 整理时间：2026-06-07

## 零、核心物理参数定义

理解所有二极管差异之前，先定义四个关键物理量：

|参数|符号|物理含义|
|-|:-:|-|
|**正向压降**|VF|PN 结/肖特基结的势垒电压 + 体电阻压降|
|**反向恢复时间**|trr|正向导通时存储的少数载流子被抽出的时间|
|**结电容**|Cj|耗尽层宽度决定的等效电容，Cj ∝ 1/VR^m|
|**击穿电压**|VB|反向电压超过此值时电流急剧增大|

**所有二极管的功能差异，本质上来自对这四个参数的不同优化方向。**

## 二、整流/开关类二极管

### 2.1 标准恢复二极管 (Standard Recovery Rectifier)

**结构**：P⁺ — N⁻ — N⁺ (PIN 或 PN 结构)

**关键参数**：

|参数|典型值|
|-|:-:|
|VF|0.7 \~ 1.0V|
|trr|1 \~ 20 μs|
|VB|50 \~ 2000V|

**物理机制**：

双极型导电——PN 结导通时，P 区注入空穴进入 N 区，N 区注入电子进入 P 区。这些少数载流子在结两侧形成存储电荷。关断时需要将这些存储电荷全部"抽出"——这就是 trr 的来源。电压等级越高 → N⁻ 漂移区越宽 → 存储电荷越多 → trr 越长。

**应用场景**：

* **工频整流 (50/60Hz)**：AC-DC 前端整流桥，用于家电、充电器输入级
* **低频逆变器**：输出频率 < 1kHz 的逆变器续流
* **汽车发电机整流**：交流发电机输出整流

**代表型号**：

|型号|VB|IF(AV)|VF @ IF|trr|封装|典型用途|
|-|:-:|:-:|:-:|:-:|-|-|
|**1N4007**|1000V|1A|1.0V|\~2μs|DO-41|通用工频整流|
|**1N5408**|1000V|3A|1.0V|\~5μs|DO-201AD|较大功率工频|
|**GBJ2510**|1000V|25A|0.9V|\~10μs|GBJ (桥堆)|大功率 AC-DC 整流桥|
|**1N4148**|100V|200mA|0.72V|4ns|SOD-123|小信号开关 (不算标准恢复，但极常用)|

### 2.2 快恢复二极管 (FRD — Fast Recovery Diode)

**结构**：PN 结 + 掺金(Au)或铂(Pt)，或电子辐照

**关键参数**：

|参数|典型值|
|-|:-:|
|VF|1.0 \~ 1.5V|
|trr|100 \~ 500 ns|
|VB|200 \~ 1200V|

**物理机制 — 为什么 trr ↓ 必须以 VF ↑ 为代价**：

```
金掺杂的效果：
  · 在硅禁带中央 (≈0.54eV from Ec) 引入深能级陷阱 → 复合中心
  · 存储的少数载流子通过陷阱快速复合 → trr ↓
  · 但这些陷阱也散射多数载流子 → 迁移率 μ ↓
  · 体电阻 ρ = 1/(qμn) ↑ → VF ↑

物理矛盾：
  trr 需要高浓度复合中心 ⇔ VF 需要高迁移率
  → 两者不可兼得，折中取决于应用频率
```

**应用场景**：

* **开关电源输出整流 (100\~300kHz)**：65W 以下反激电源的次级整流
* **PFC 升压二极管**：需要 600V 以上耐压 + 较快恢复
* **IGBT 续流二极管**：变频器、电机驱动中的反并联二极管
* **电焊机输出整流**：高频逆变电焊机次级

**代表型号**：

|型号|VB|IF(AV)|VF @ IF|trr|典型用途|
|-|:-:|:-:|:-:|:-:|-|
|**FR107**|1000V|1A|1.3V|500ns|小功率反激输出整流|
|**FR307**|1000V|3A|1.3V|500ns|中功率反激/正激输出|
|**STTH8R06**|600V|8A|1.4V|25ns|PFC 升压二极管 (DCM)|
|**RHRP1560**|600V|15A|1.5V|45ns|大功率 PFC/变频器续流|

### 2.3 超快恢复二极管 (UFRD — Ultra-Fast Recovery Diode)

**结构**：PN 结 + 重金掺杂 + 外延工艺优化

**关键参数**：

|参数|典型值|
|-|:-:|
|VF|1.2 \~ 2.0V|
|trr|25 \~ 100 ns|
|VB|200 \~ 1200V|

**软恢复 vs 硬恢复波形对比**：

```
硬恢复 (普通快恢复):            软恢复 (超快恢复):

 IF                               IF
  │  ╲                             │    ╲
  │   ╲                            │     ╲\_\_\_
  │    ╲\_\_\_                        │        ╲\_\_\_
  │       ╲\_\_\_                     │           ╲\_\_\_
  └──────────── t                  └───────────── t
    ↑ 尖峰 + 振铃                     ↑ 平滑过渡
  di/dt 大 → V=L×di/dt 大          di/dt 小 → EMI 低
  → 需要 snubber 吸收              → 可能省掉 snubber

软恢复原理：
  芯片中心区域复合中心浓度高(快速抽出) + 边缘浓度低(缓慢过渡)
  → 避免电流突然截断(di/dt)造成的电压尖峰
```

**应用场景**：

* **高频开关电源 (300kHz\~1MHz)**：大功率通信电源
* **服务器电源输出整流**：LLC 谐振拓扑同步整流前级
* **光伏逆变器 Boost 二极管**：输入 >300V，开关频率 16\~50kHz
* **电动汽车 OBC (车载充电机)**：PFC 输出整流

**代表型号**：

|型号|VB|IF(AV)|VF @ IF|trr|典型用途|
|-|:-:|:-:|:-:|:-:|-|
|**MUR860**|600V|8A|1.5V|60ns|服务器电源输出整流|
|**MUR1560**|600V|15A|1.5V|60ns|大功率 PFC/变频器|
|**STTH3002**|200V|30A|0.85V|22ns|低压大电流高频整流|
|**FFPF60UP20DN**|200V|60A|0.95V|45ns|大电流续流/电焊机|

### 2.4 肖特基二极管 (Schottky Diode)

**结构**：金属 (Pt, Mo, Cr, Ti, Ni) — N⁻ — N⁺

```
     金属       N⁻ 半导体        N⁺ 衬底
     │    ←── 肖特基结 ──→    │    ←── 欧姆接触 ──→
     │                        │
   阳极      耗尽层            阴极
```

**关键参数**：

|参数|典型值|与 PN 结对比|
|-|:-:|-|
|VF|0.15 \~ 0.45V|PN 结: 0.7\~1.0V|
|trr|≈ 0 (ps 级)|PN 结快恢复: 25\~500ns|
|VB|20 \~ 200V|PN 结: 可达 2000V+|
|漏电流 IR|μA \~ mA 级|PN 结: nA \~ μA|

**肖特基 vs PN 结导电机制对比**：

```
PN 结导通 (双极型导电):          肖特基结导通 (单极型导电):

  P⁺            N⁻               Metal          N⁻
  ─→ 空穴       ←─ 电子            ─→ 电子        ←─ 电子
  ←─ 电子       ─→ 空穴           (只有多数载流子!)
  (两种载流子参与)                  (只有电子参与)

PN 结关断时：
  储存在 N 区的大量空穴需要被"抽回"P 区
  → 需要时间 → trr

肖特基关断时：
  不涉及少数载流子的注入和抽取
  → 仅需对结电容充放电 → trr ≈ 0 (理论 ps 级)
```

**SiC 肖特基 — 宽禁带突破**：

```
SiC (碳化硅) 肖特基:
  · 禁带宽度 Eg ≈ 3.26eV         (vs Si 1.12eV)
  · 临界击穿电场 ≈ 2.8MV/cm      (vs Si 0.3MV/cm)   → 高 \~10×
  · 热导率 ≈ 4.9 W/(cm·K)        (vs Si 1.5)        → 好 \~3×

  → N⁻ 漂移区可薄 10 倍 → 电阻低 100 倍 (相同 VB)
  → VB 可达 650V \~ 1700V (而 Si 肖特基 ≤ 200V)
  → trr 依然 ≈ 0 (肖特基的本质特征不变)
```

**应用场景**：

**低压 Si 肖特基 (≤ 200V)**：

* **DC-DC 续流二极管**：Buck 电路 catch diode
* **同步整流 MOSFET 体二极管替代**：死区时间内续流，VF=0.3V 大幅降低死区损耗
* **OR-ing 二极管**：多电源冗余系统中防止反向馈电
* **反接保护**：输入串联，VF 仅 0.3V
* **Flyback 输出整流 (<100V)**：VF 低 = 效率大幅提升
* **太阳能旁路二极管**：光伏组件中防止阴影热斑
* **RF 检波/混频**：低势垒肖特基 (VF ≈ 0.15V)

**高压 SiC 肖特基 (≥ 600V)**：

* **PFC 升压二极管**：大功率通信电源，效率提升 1\~2%
* **光伏逆变器 Boost 二极管**：trr≈0 消除反向恢复损耗
* **电动汽车 OBC PFC 输出整流**
* **充电桩 DC 模块输出整流**

|-|-|:-:|:-:|:-:|-|
|Si 小电流|**SS34**|40V|3A|0.5V|Buck 续流 (TPS5430 同类)|
|Si 小电流|**1N5822**|40V|3A|0.525V|低压 DC-DC 续流|
|Si 中电流|**MBR2045CT**|45V|20A|0.57V|大电流 Buck/反激输出|

## 三、稳压/保护类二极管

### 3.1 齐纳二极管 (Zener Diode)

**结构**：重掺杂 P⁺ — N⁺，极窄耗尽层 → 极高电场

**击穿机制 vs 电压范围**：

|击穿类型|电压范围|温度系数|物理本质|
|-|:-:|:-:|-|
|齐纳击穿|< 5V|**负** (dVZ/dT < 0)|量子隧穿：耗尽层 < 10nm，电子直接隧穿禁带|
|混合区|5\~6V|≈ 0|齐纳 + 雪崩并存|
|雪崩击穿|> 6V|**正** (dVZ/dT > 0)|碰撞电离：载流子加速撞击晶格，链式倍增|

**温度系数物理图解**：

```
dVZ/dT
  │
  │ 正温漂
  │    ╱ (雪崩区: 散射增强 → 需要更高电压)
  │   ╱
  │  ╱
  ├─★──────────────────── VZ
  │  ╲  ← 5.6V ≈ 零温漂点（"甜点"）
  │   ╲
  │    ╲ (齐纳区: Eg↓ → 隧穿↑ → VZ↓)
  │     负温漂
  │
```

**应用场景**：

* **电压基准**：5.6V/6.2V 齐纳 (≈ 零温漂)，用于线性电源、仪器仪表的基准源
* **过压钳位**：并联在敏感负载两端
* **栅极过压保护**：MOSFET 栅极并联 12\~15V 齐纳
* **电平移位**：两个不同电压域的接口转换
* **白噪声发生器**：齐纳击穿时的散粒噪声 → 密码学随机数生成

**代表型号**：

|-|:-:|:-:|:-:|:-:|-|
|**BZX55C5V6**|5.6V|±5%|500mW|≈ 0|电压基准/参考|
|**BZV55C12**|12V|±5%|500mW|+7mV/K|栅极保护/过压钳位|
|**1N4733A**|5.1V|±5%|1W|±0.03%/K|通用稳压|
|**1N4742A**|12V|±5%|1W|+0.07%/K|12V 电源过压钳位|
|**BZX84C3V3**|3.3V|±5%|250mW|−0.06%/K|3.3V 逻辑过压保护|

### 3.2 TVS 二极管 (Transient Voltage Suppressor)

**结构**：本质是经过特殊工艺优化的大面积雪崩 PN 结，截面积极大 (mm² 级)，金线键合，低寄生电感封装。

**与普通齐纳的关键差异**：

|参数|TVS|普通齐纳|
|-|:-:|:-:|
|峰值脉冲功率|**400W \~ 30kW**|mW \~ 1W|
|响应时间|**< 1ps** (理论) / < 1ns (实际)|ns 级|
|结面积|数 mm²|\~0.01mm²|
|钳位比 VCL/VBR|1.2 \~ 1.5|> 2|
|结电容|50 \~ 5000pF|\~50pF|

**双向 TVS 结构**：

```
单向 TVS:                        双向 TVS:

  ──├──▶├──                        ──├──▶├──┬──▶├──├──
    │                            (背靠背雪崩管)
    └── 地                          └────────┘

正压: 雪崩导通                    正压: A 管雪崩, B 管正向导通
负压: 正向导通 (≈0.7V)            负压: B 管雪崩, A 管正向导通

使用场景: DC 电源线              使用场景: AC/差分信号线
```

**应用场景**：

* **电源入口保护**：汽车抛负载 (Load Dump: 12V→80V/400ms)、USB VBUS 热插拔浪涌、24V 工业感性负载关断、PoE 48V
* **信号线 ESD 保护**：HDMI/USB3.0 差分对 (超低电容)、RS-485 总线 (共模浪涌)、CAN 总线 (汽车标配)
* **雷击浪涌保护**：电信线入口 (Tip/Ring) 大功率 TVS、与 GDT/MOV 多级配合
* **GaN/SiC 栅极保护**：VGS\_max ≈ ±6\~10V，需精确低压 TVS

**选型要点**：

```
① VRWM > 电路最大工作电压 (有裕量)
② VBR 略高于 VRWM
③ VCL < 被保护器件的最大耐受电压 (这是最关键的一条!)
④ PPP > 预期的浪涌能量
⑤ 信号线需关注结电容
    · 低速 (< 1Mbps): Cj < 50pF 即可
    · USB 2.0 (480Mbps): Cj < 3pF
    · HDMI 2.0 (6Gbps): Cj < 0.5pF
```

## 四、RF/高频类二极管

### 4.1 变容二极管 (Varactor / Varicap)

**结构**：特殊掺杂分布的反偏 PN 结，耗尽层宽度由反压控制。

**物理原理**：

```
结电容（平行板电容器模型）：

         ε × A
Cj = ────────────     (ε=介电常数, A=结面积, W=耗尽层宽度)
          W(V)

掺杂分布决定电容-电压关系：
  · 突变结 (Abrupt):  W ∝ √(VR)   → Cj ∝ 1/VR^(1/2)        → 调谐比约为 3\~4
  · 缓变结 (Graded):  W ∝ ∛(VR)   → Cj ∝ 1/VR^(1/3)        → 调谐比约为 2\~3
  · 超突变结 (Hyper): Cj ∝ 1/VR^m (m > 1/2)                → 调谐比可达 10\~15
```

**等效电路 (反偏)**：

```
  ──┬──── Rs ────┬────
    │  (0.2\~2Ω)  │
    │            ├── Cj(V) ──┤
    │            │  (可变电容)
    │            ├── Rj ──┤
    │            │  (> MΩ, 可忽略)
    └────────────┴────

Q 值 = 1/(2πf × Rs × Cj)
高 Q 值 → 低损耗 → 优良的调谐元件
```

**应用场景**：

* **VCO (压控振荡器)**：PLL 锁相环核心，手机 RF 收发器、WiFi 芯片本地振荡器
* **天线阻抗调谐**：手机天线匹配网络，LTE Band 1\~41 动态调谐
* **电调滤波器**：替代机械调谐的可调带通滤波器
* **FM 收音机调谐**：经典应用
* **参量放大器 (Parametric Amp)**：射电天文望远镜极低噪声微波放大

**代表型号**：

|型号|Cmax/Cmin|VR 范围|C3V (pF)|典型用途|
|-|:-:|-|:-:|-|
|**BBY40**|\~6|1\~25V|26|VCO、天线调谐|
|**BB135**|\~9|0.5\~28V|20|FM 收音机调谐|
|**SMV1231**|\~3.5|0\~15V|2.35|UHF/VHF 调谐|
|**BBY58-02V**|\~3|1\~10V|5.5|手机天线孔径调谐|

### 4.2 PIN 二极管

**结构**：P⁺ — I (本征/轻掺杂) — N⁺，I 层宽度 10 \~ 200μm。

**核心特性**：

|偏置状态|低频行为|高频行为 (> fC)|
|-|-|-|
|正向|导通 (像普通 PN)|**可变电阻**，R ∝ 1/IF|
|反向|截止 (高阻)|**小电容**，C ∝ 1/W\_I|

**等效电路随频率变化**：

```
低频 (< fC):                     高频 (> fC):

  ──┤← 普通 PN 结 →├──            ──┤ R(IF) ├──┤ C ├──
    (导通或截止)                    (偏置可控电阻 + 电容)
                                   
  其中 fC = 1/(2πτ), τ = 少数载流子寿命
  物理: RF 周期 << I 层渡越时间 → 载流子浓度在每个 RF 周期内不变
```

* **RF 限幅器**：雷达接收机前端保护

## 六、特殊器件

### 6.1 隧道二极管 (Tunnel Diode / Esaki Diode)

```
结构: 极重掺杂 P⁺⁺ — N⁺⁺，耗尽层 < 5nm

IV 特性:
 IF
  │
  │  ╱╲
  │ ╱  ╲  ← 负阻区 (隧道效应衰退)
  │╱    ╲
──┼──────┼────── VF
  │  VP  VV
  
  VP = 峰值电压 (\~50\~100mV)
  VV = 谷值电压 (\~300\~500mV)
  IP/IV = 峰谷比 (典型 5\~20)

物理:
  · 极重掺杂 → 耗尽层 < 5nm → 电子量子隧穿通过势垒
  · 低电压时隧穿概率高 → 大电流 (峰值点)
  · 电压升高 → 能带错位 → 隧穿概率下降 → 电流减小 (负阻区)
  · 高电压 → 恢复正常热电子发射 (与普通 PN 结相同)
```

**应用场景**：

* **超高速开关**：皮秒级开关速度，取样示波器采样门
* **微波振荡器**：利用负阻特性产生 GHz 级振荡
* **触发器**：超快双稳态电路

> 注：隧道二极管在现代电路中使用较少，已被高速晶体管 (SiGe HBT, InP HEMT) 部分替代。

### 6.2 Gunn 二极管 (Gunn Diode / Transferred Electron Device)

```
结构: N⁺ — N (GaAs) — N⁺，没有 PN 结，纯体效应器件

物理机制 (Gunn 效应):
  GaAs 导带有两个能谷:
    · Γ 谷 (低能): 电子有效质量小 → 迁移率高 (μ ≈ 8000 cm²/V·s)
    · L 谷 (高能): 电子有效质量大 → 迁移率低 (μ ≈ 100 cm²/V·s)

  低电场: 电子在 Γ 谷 → 高迁移率 → 高漂移速度
  高电场 (>3.5kV/cm): 电子获得足够能量，跃迁至 L 谷
    → 迁移率骤降 80 倍 → 漂移速度反而减小
    → 负微分迁移率 → 产生电荷畴 (domain)
    → 畴在电场下从阴极向阳极漂移 → 到达阳极时被吸收
    → 新畴在阴极再次形成 → 周期性 → 电流振荡

振荡频率: f ≈ v\_drift/L (10\~100GHz)
```

**应用场景**：

* **微波/毫米波振荡器**：汽车防撞雷达 (77GHz)、安检成像
* **速度传感器**：配合 Doppler 雷达测速
* **微波信号源**：实验室微波测试设备

## 七、选型决策树

```
需要二极管？
  │
  ├─ 整流/续流？
  │    ├─ 电压 > 200V？
  │    │    ├─ 频率 > 100kHz？ → SiC 肖特基 (>600V) 或超快恢复 (300V\~600V)
  │    │    └─ 频率 < 1kHz？ → 标准恢复（便宜, VF 低）
  │    ├─ 频率 > 300kHz？
  │    │    ├─ 电压 < 200V → 肖特基（VF 低 + trr≈0 双重优势）
  │    │    └─ 电压 > 200V → 超快恢复或 SiC 肖特基
  │    └─ 工频 50/60Hz → 标准恢复（最具性价比）
  │
  ├─ 稳压/保护？
  │    ├─ 持续稳压（mW 级） → 齐纳 (5.6V 推荐，≈ 零温漂)
  │    ├─ 瞬态保护：
  │    │    ├─ 电源入口浪涌 (kW 级) → TVS (5kW 系列)
  │    │    ├─ ESD 保护 (I/O 口) → 低电容 TVS 阵列
  │    │    ├─ 信号线浪涌 → 双向 TVS
  │    │    └─ 雷击 → TVS + GDT/MOV 多级配合
  │    └─ MOSFET 栅极保护 → 低压齐纳或低压 TVS
  │
  ├─ RF/高频？
  │    ├─ 需要可变电容 → 变容管 (BBY40, BB135)
  │    ├─ RF 开关/衰减 → PIN 管 (频率 > 10MHz 时使用)
  │    ├─ 检波/混频 → 低势垒肖特基 (VF ≈ 0.15V)
  │    └─ 限幅/保护 → PIN 限幅器
  │
  ├─ 光电？
  │    ├─ 发光：
  │    │    ├─ 指示/照明 → LED
  │    │    ├─ 光纤通信 → DFB 激光器 / VCSEL
  │    │    ├─ 红外发射 → IR LED (GaAs, 870nm)
  │    │    └─ LiDAR → VCSEL 阵列 / 边发射激光器
  │    └─ 检测/接收：
  │         ├─ 可见光/近红外 → Si PIN PD
  │         ├─ 光纤通信 (1310/1550nm) → InGaAs PIN PD / APD
  │         ├─ 紫外检测 → GaP / SiC PD
  │         ├─ 单光子级别 → APD / SiPM (SPAD)
  │         └─ 光耦隔离反馈 → Si PD (集成在光耦中)
  │
  └─ 特殊应用？
       ├─ 微波振荡 (>10GHz) → Gunn 二极管
       ├─ ps 级开关/负阻 → 隧道二极管
       └─ 射频能量采集 → 低势垒肖特基检波管
```
