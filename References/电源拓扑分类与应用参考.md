# 电源拓扑分类与应用参考

> 整理时间：2026-06-07

\---

## 零、总览：从线性到谐振

```
                        电能变换拓扑
                            │
        ┌───────────────────┼───────────────────┐
        │                   │                   │
    线性稳压            开关稳压 (PWM)         谐振变换
        │                   │                   │
    ·标准 LDO          ┌────┼────┐           ·LLC
    ·准 LDO          非隔离  隔离            ·串联谐振
    ·VLDO               │     │             ·并联谐振
                    ┌───┼───┐ │             ·LLCC
                    │   │   │ │
                  Buck Boost│ Flyback
                        │   │ Forward
                     Buck-  │ Push-Pull
                     Boost  │ Half-Bridge
                     SEPIC  │ Full-Bridge
                     Ćuk    │

变换本质: 功率 = V × I，拓扑定义了 V、I 之间的转换关系
```

### 选型第一原则：隔离 vs 非隔离

```
需要隔离？
  │
  ├─ 否 → 非隔离拓扑
  │       ├─ VOUT < VIN → Buck
  │       ├─ VOUT > VIN → Boost
  │       └─ VOUT ⪋ VIN → Buck-Boost / SEPIC
  │
  └─ 是 → 隔离拓扑
          ├─ < 100W  → Flyback
          ├─ 100\~500W → Forward / Push-Pull
          └─ > 500W  → Half-Bridge / Full-Bridge / LLC
```

## 一、线性稳压器 (Linear Regulator)

### 1.1 基本原理

```
        ┌─── 调整管 (BJT/MOSFET) ───┐
VIN ───┤  工作在线性区            ├───┬─── VOUT
        │  (像个可变电阻)          │   │
        └───────────────────────┘   │
                              ┌─────┘
                       误差放大器 ← VREF ─┤
                              │
                             GND
```

调整管工作在线性区（非开关状态），通过反馈控制等效阻抗，将 VIN−VOUT 的压差"消耗"在调整管上。

### 1.2 核心公式

```
VOUT = VREF × (1 + R1/R2)

效率:
η = VOUT / VIN × 100%    (理想情况，忽略静态电流)

功率损耗 (调整管):
PD = (VIN − VOUT) × ILOAD

压差 (Dropout Voltage):
VDROPOUT = VIN(min) − VOUT = ILOAD × RDS(on)    (MOSFET LDO)
```

### 1.3 分类

|类型|调整管|压差|静态电流|典型应用|
|-|-|:-:|:-:|-|
|**标准 LDO**|PNP BJT|0.5\~1.5V|mA 级|通用稳压|
|**准 LDO**|NPN BJT|0.3\~1.0V|mA 级|中功率稳压|
|**P-MOS LDO**|P-Ch MOSFET|50\~500mV|μA 级|电池供电设备|
|**N-MOS LDO**|N-Ch MOSFET|30\~200mV|μA 级|大电流、低噪声|

### 1.4 优缺点

|优点|缺点|
|-|-|
|输出噪声极低 (μV 级)|效率低 (VIN≫VOUT 时严重)|
|无 EMI/开关噪声|只能降压|
|外围元件极少|大功率需散热器|
|瞬态响应快|功率密度低|
|成本极低||

### 1.5 典型应用

* RF/模拟前端供电（PLL、ADC、运放）
* 后级稳压（开关电源 + LDO）
* 电池供电低功耗设备
* 对噪声敏感的传感器

## 二、Buck (降压) 变换器

### 2.1 基本拓扑

```
       ┌── S ──┬── L ──┬── VOUT
VIN ──┤        │        │
       │       ├── D ───┤
       └───────┘        ├── COUT
                        │
                        └── GND
S = 开关管 (MOSFET)
D = 续流二极管 (或同步整流 MOSFET)
```

### 2.2 工作原理

```
Phase 1 (S ON):     Phase 2 (S OFF):

VIN ──→ L ──→ COUT   L ──→ COUT
        ↓ 储能          ↑ 释能 (di/dt = −VOUT/L)
    di/dt = (VIN−VOUT)/L  │
                      D ← 续流
```

### 2.3 核心公式

```
占空比:
D = VOUT / VIN         (CCM, 理想)

电感电流纹波:
ΔIL = VOUT × (VIN−VOUT) / (VIN × L × fSW)

CCM/DCM 边界:
IOUT(crit) = ΔIL / 2 = VOUT × (1−D) / (2 × L × fSW)

输出电压:
VOUT = D × VIN         (CCM)

开关管电压应力: VDS(max) = VIN(max)
二极管电压应力: VD(max) = VIN(max)
```

### 2.4 同步整流 Buck

```
       ┌── S1 (高侧) ──┬── L ──┬── VOUT
VIN ──┤                 │        │
       │                ├── S2 ──┤     ← 用 MOSFET 替代二极管
       └────────────────┘        ├── COUT
                                 │
                                 └── GND
S2 在 S1 关断时导通 → VF 从 \~0.5V (肖特基) 降至 \~0.1V (RDS(on)×I)
→ 效率大幅提升 (尤其低压大电流)
→ 缺点: 需要死区控制防止穿通
```

### 2.5 优缺点

|优点|缺点|
|-|-|
|高效率 (85\~95%)|只能降压|
|输出电流大|输入端需处理脉冲电流|
|简单、成本低|MOSFET 高侧驱动需自举|
|适合低压大电流||

### 2.6 典型应用

* **5V/3.3V/1.8V 板上供电**（从 12V/24V 降压）
* **POL (Point of Load)**：CPU/FPGA/ASIC 核心供电，12V→0.8V/100A
* **电池充电器 (CC/CV Buck)**
* **LED 恒流驱动**（Buck 模式）
* **USB PD 受电端**（20V→5V/3A）

## 三、Boost (升压) 变换器

### 3.1 基本拓扑

```
          L         D
VIN ──┬── 000 ──┬──▶├──┬── VOUT
      │         │       │
      │        ┌┘       ├── COUT
      │        │S       │
      └────────┴────────┴── GND
```

### 3.2 工作原理

```
Phase 1 (S ON):     Phase 2 (S OFF):

VIN ──→ L ──→ S      VIN ──→ L ──→ D ──→ VOUT
        ↓ 储能               ↓ 释能 + VIN 叠加
    di/dt = VIN/L          VOUT = VIN + VL
```

### 3.3 核心公式

```
占空比:
D = (VOUT − VIN) / VOUT = 1 − VIN/VOUT     (CCM)

电感电流纹波:
ΔIL = VIN × D / (L × fSW)

输出电压:
VOUT = VIN / (1−D)         (CCM)
当 D → 1 时 VOUT → ∞ (理论), 实际受限于寄生参数

CCM/DCM 边界:
IOUT(crit) = VOUT × D × (1−D)² / (2 × L × fSW)

开关管电压应力: VDS(max) = VOUT
二极管电压应力: VD(max) = VOUT

输入电流 (平均值):
IIN(avg) = IOUT / (1−D)    > IOUT

电感电流 (平均值) = IIN(avg)   (电感在输入回路中)
```

### 3.4 右半平面零点 (RHPZ)

Boost 在 CCM 下存在**右半平面零点**，是控制上的核心难题：

```
fRHPZ = (1−D)² × RLOAD / (2π × L)

物理含义:
  负载加重 → 占空比需要增大 → 但增大 D 先让电感储能时间变长
  → 短期内传递到输出的能量反而减少 → VOUT 先跌后升
  → 非最小相位系统 → 补偿困难 → 带宽受限于 fRHPZ/3

应对: 穿越频率 fc < fRHPZ/3, 通常 fc < 1\~2kHz (高升压比时很慢)
```

### 3.5 优缺点

|优点|缺点|
|-|-|
|只需一个开关管|右半平面零点 (难补偿)|
|只能升压|启动时有浪涌电流路径 (VIN→L→D→COUT)|
|效率高|输出短路保护困难|
|成本低|轻载时如果不跳周期，效率低|

### 3.6 典型应用

* **升压型 LED 背光驱动**（3.7V→12\~40V）
* **USB PD 供电端**（5V→9V/15V/20V）
* **电池升压**（单节锂电 3.7V→5V）
* **PFC 前级**（整流后 Boost 型 PFC）
* **闪光灯电容充电**
* **汽车冷启动升压保持**

## 四、Buck-Boost (升降压) 变换器

### 4.1 反相 Buck-Boost (Inverting)

```
           S          D
VIN ───┬──/ ──┬───▶├───┬── VOUT (−)
       │      │        │
       └──────┘        ├── COUT
       L               │
                       └── GND (负载参考 = VIN)
VOUT 极性: 与 VIN 相反 (负输出)
```

### 4.2 工作原理

```
Phase 1 (S ON):     Phase 2 (S OFF):
VIN ──→ S ──→ L ──→ GND    L ──→ D ──→ COUT − (负端)
           ↓ 储能              ↑ 释能
    di/dt = VIN/L          感应电势反转, 对 COUT 反向充电
```

### 4.3 核心公式

```
占空比:
D = VOUT / (VOUT − VIN)          (注意: VOUT 取绝对值)

输出电压:
|VOUT| = VIN × D/(1−D)

当 D<0.5: |VOUT| < VIN  (降压模式)
当 D>0.5: |VOUT| > VIN  (升压模式)

开关管电压应力:
VDS(max) = VIN + |VOUT|

二极管电压应力: VD(max) = VIN + |VOUT|

电感电流 (平均值):
IL(avg) = IOUT × (1 + |VOUT|/VIN)
```

### 4.4 四开关 Buck-Boost (Non-Inverting)

```
        S1      S3
VIN ───/ ──┬───/ ──┬── VOUT (+)  (正输出)
           │       │
           L       COUT
           │       │
           └───────┴── GND
        S2      S4

控制模式:
  · VIN > VOUT → Buck 模式   (S1 PWM, S3 常通, S2/S4 互补)
  · VIN < VOUT → Boost 模式  (S1 常通, S3 PWM, S2/S4 互补)
  · VIN ≈ VOUT → Buck-Boost  (四管全切换, 效率最低)
```

### 4.5 优缺点

|优点|缺点|
|-|-|
|可升可降|反相型: 输出负压|
|单电感|开关/二极管应力 = VIN+|
|适合宽输入电压|效率略低于纯 Buck/Boost|
|四开关型: 正输出|四开关: 4个 MOSFET, 控制复杂|

### 4.6 典型应用

* **电池供电设备**（锂电 2.7\~4.2V → 3.3V）：输入可高可低
* **汽车电子**（冷启动 4V \~ 正常 14V → 12V/5V）
* **USB 供电设备**（VBUS 5V → 3.3V，四开关 Buck-Boost）
* **负压偏置**（GaN PA 栅极、运放负电源）

## 五、SEPIC (单端初级电感变换器)

### 5.1 基本拓扑

```
         Cc (耦合电容)
VIN ───┬──┤├──┬── D ──┬── VOUT
       │      │       │
       L1     L2      COUT
       │      │       │
       └── S ─┴───────┴── GND

L1 和 L2 可耦合 (1个磁芯) 或分立 (2个电感)
```

### 5.2 工作原理

```
Phase 1 (S ON):             Phase 2 (S OFF):
VIN ──→ L1 ──→ S             L1 ──→ Cc ──→ D ──→ COUT
Cc ──→ L2 ──→ S             L2 ──→ D ──→ COUT
(两个电感同时储能)            (两个电感同时释能)
```

### 5.3 核心公式

```
占空比:
D = VOUT / (VOUT + VIN)

输出电压:
VOUT = VIN × D/(1−D)       (正输出，同 Buck-Boost 公式)

耦合电容 Cc 的选择:
Cc 需要处理全部输入电流 → 必须选低 ESR 陶瓷
电压应力: VCc = VIN
纹波电流: ICc(RMS) ≈ IOUT × √(D/(1−D))
```

### 5.4 SEPIC vs Buck-Boost

||SEPIC|反相 Buck-Boost|
|-|-|-|
|输出极性|**正**|负|
|开关应力|VIN+VOUT|VIN+VOUT|
|EMI|输入电流连续 (L1)|输入电流脉冲|
|元件数|2电感 + 1耦合电容|1电感|
|效率|略低 (电容 ESR 损耗)|略高|

### 5.5 典型应用

* **汽车电子** (宽输入 4\~40V → 12V)
* **电池供电设备** (锂电→5V/3.3V)
* **LED 驱动器** (需要升降压恒流)

## 六、Ćuk 变换器

### 6.1 基本拓扑

```
         Cc
VIN ───┬──┤├──┬── L2 ──┬── VOUT (−)
       L1     D        │
       │      │        COUT
       │      └── S ───┤
       └───────────────┴── GND

与 SEPIC 对偶: D 和 S 位置互换
输出极性: 与 VIN 相反 (负输出)
```

### 6.2 核心特点

```
输入电流连续 (L1) + 输出电流连续 (L2)
→ 输入/输出纹波都极小
→ EMI 特性极好

代价:
  · 输出负压 (可用耦合电感实现正输出)
  · 耦合电容需承受全部负载电流
  · DCR 损耗 ×2 (两个电感)
```

### 6.3 典型应用

* **低噪声/低 EMI 电源** (对纹波极敏感)
* **负压电源**
* **电池充电器**

\---

## 七、Flyback (反激) 变换器

### 7.1 基本拓扑

```
       ┌── Np ──┬─ 磁芯 ─┬── Ns ── D ──┬── VOUT
VIN ──┤         │   │    │             │
       │         │   │ ← 气隙           ├── COUT
       └── S ───┘   │    └─────────────┤
                     │                  └── GND (次级)
              隔离边界  (变压器)

本质: 带变压器的 Buck-Boost
```

### 7.2 工作原理

```
Phase 1 (S ON):
  初级 Np 储能 (变压器作为电感)
  次级 D 反偏 → 不导通
  负载由 COUT 供电

Phase 2 (S OFF):
  磁芯释能 → 次级 Ns 感应电压 → D 导通
  → 能量传递到负载 + 补充 COUT

核心特点: 变压器储能在开关 ON 时 → OFF 时传递到次级
         (这与 Forward 完全不同!)
```

### 7.3 核心公式

```
占空比 (CCM):
D = (VOUT + VF) × (Np/Ns) / \[VIN + (VOUT + VF) × (Np/Ns)]

匝比:
n = Np/Ns = VIN(min) × Dmax / \[(VOUT + VF) × (1−Dmax)]

Dmax 通常取 0.45\~0.5 (留余量给漏感复位)

初级电感 (CCM):
Lp(min) = VIN(min)² × Dmax² / (2 × PIN × fSW × KRF)

KRF = ΔIL/IL(peak) ≈ 0.3\~0.6 (纹波系数)

开关管电压应力:
VDS(max) = VIN(max) + VCLAMP + Vspike
VCLAMP = (VOUT + VF) × Np/Ns + Vleakage\_spike (\~30%)
→ 通常 VDS(max) ≈ VIN(max) + VOR + Vspike

次级二极管电压应力:
VD(max) = VOUT + VIN(max) × Ns/Np
```

### 7.4 RCD 钳位 (吸收漏感能量)

```
         Rclamp
          │
VIN ──┬──┤├──┬── Np ──┬── ...
      │  Cclamp│       │
      │       └── D ───┤
      └── S ───────────┘

漏感能量: Eleak = 0.5 × Lleak × Ipeak²
钳位电容: Cclamp 吸收漏感尖峰
Rclamp 消耗这部分能量

Vclamp = VOR + ΔVclamp
ΔVclamp ≈ VOR × 0.3
```

### 7.5 CCM vs DCM 对比

```
CCM (连续模式):                   DCM (断续模式):

IL                              IL
│  ╱╲  ╱╲  ╱╲                  │  ╱╲        ╱╲
│ ╱  ╲╱  ╲╱  ╲                 │ ╱  ╲      ╱  ╲
│╱            ╲                │╱    ╲    ╱    ╲
──────────────── t              │      ╲  ╱      ╲
  不归零                            ╲╱        ╲╱
                                    ───────────── t
                                    归零 (死区)

CCM: 导通损耗小, 峰值电流小 → 适合大功率 (但补偿困难, 有 RHPZ)
DCM: ZVS 开启 (谷底开关), 次级二极管无 trr → 适合小功率/高频
```

### 7.6 优缺点

|优点|缺点|
|-|-|
|单开关管|变压器需气隙 (储能)|
|输入输出隔离|漏感 → EMI + 效率损失|
|宽输入电压范围|输出纹波大|
|可多路输出|大功率时变压器体积大|
|成本最低的隔离方案|RHPZ (CCM 时)|

### 7.7 典型应用

* **AC-DC 适配器** (< 100W)：手机充电器、笔记本适配器
* **多路输出辅助电源**：从主电源取一路做辅助供电
* **USB PD 充电器** (20\~100W)
* **LED 隔离驱动器**
* **POE (以太网供电) PD 端**

## 八、Forward (正激) 变换器

### 8.1 基本拓扑 (单端正激)

```
            复位绕组 Nr
             │
VIN ──┬── Np ─┬── Nr ── D\_reset ── VIN
      │       │
      └── S ──┘

      Ns ─── D1 ── L ──┬── VOUT
      │          │      │
      │          D2     COUT
      └──────────┤      │
                 └──────┴── GND (次级)
D1 = 整流管, D2 = 续流管 (与 Buck 的 D 作用相同)
```

### 8.2 工作原理

```
Phase 1 (S ON):
  能量直接从初级经变压器传递到次级
  D1 导通 → L 储能 → COUT 充电
  变压器不储能! (仅传递能量)

Phase 2 (S OFF):
  磁芯复位: Nr 将励磁能量返回 VIN
  D1 截止, D2 续流 → L 释能给负载

关键区别 (Flyback vs Forward):
  Flyback: 变压器储存能量 → OFF 时释放
  Forward: 变压器传输能量 → ON 时传递, 需额外复位绕组
```

### 8.3 核心公式

```
占空比:
D = (VOUT + VF) × Np/(Ns × VIN)

变压器匝比:
n = Np/Ns = VIN(min) × Dmax / (VOUT + VF)

Dmax 必须 < 0.5 (对于单端复位绕组):
  励磁电流必须在 OFF 期间完全复位
  复位时间 ≤ (1−D) × Tsw
  → Dmax ≤ 0.5 (Nr=Np 时)

开关管电压应力:
VDS(max) = VIN(max) × (1 + 1/Np\_to\_Nr\_ratio)
Np = Nr → VDS(max) = 2 × VIN(max)

二极管电压应力:
VD1(max) = VIN(max) × Ns/Np
VD2(max) = VIN(max) × Ns/Np
```

### 8.4 双管正激 (Two-Switch Forward)

```
       ┌── S1 ─── Np ─── S2 ──┐
       │                      │
VIN ───┤                      ├── VIN
       │    D3       D4       │
       └────┤◀├──┬──┤◀├───────┘
                 │
                GND
复位: D3,D4 直接将励磁电流返回 VIN
VDS(max) = VIN(max) (单管正激的一半!)
不需要额外复位绕组
```

### 8.5 优缺点

|优点|缺点|
|-|-|
|LC 输出滤波 → 纹波低|需要复位绕组或双管|
|效率高于 Flyback|Dmax ≤ 0.5|
|变压器不需要气隙|磁芯利用率低 (第一象限)|
|适合中等功率|需要输出电感 (占空间)|

### 8.6 典型应用

* **ATX 电源辅助输出** (3.3V/5V)
* **通信电源** (48V→12V, 100\~300W)
* **工业电源**
* **DC-DC 隔离模块** (双管正激)

## 九、Push-Pull (推挽) 变换器

### 9.1 基本拓扑

```
             Np1 ←→ Np2 (中心抽头)
              │     │
VIN ──┬──────┤     ├──────┬──
      │      │     │      │
      │   S1 ┘     └ S2   │  ← 交替导通
      │                   │
      └────────┬──────────┘
              GND

     Ns (中心抽头)
      │
      ├── D1 ── L ──┬── VOUT
      │              │
      ├── D2 ────┤   COUT
      │              │
      └──────────────┴── GND
```

### 9.2 工作原理

```
S1 ON, S2 OFF: 电流经 Np1 → 磁芯正向励磁
               → Ns 上半 → D1 导通 → L → 负载

S2 ON, S1 OFF: 电流经 Np2 → 磁芯反向励磁
               → Ns 下半 → D2 导通 → L → 负载

磁芯 B-H 曲线: 工作在第一和第三象限 (双向励磁)
→ 磁芯利用率高 (vs Forward 只在一象限)
```

### 9.3 核心公式

```
占空比:
D = (VOUT + VF) × Np/(Ns × VIN)    (每管 D < 0.5)

两个开关交替导通, 等效占空比 = 2×D

开关管电压应力:
VDS(max) = 2 × VIN(max)
(变压器中心抽头效应 → 关断管承受 2×VIN)

输出电感纹波频率 = 2 × fSW
(推挽双向励磁, 等效频率翻倍)
```

### 9.4 优缺点

|优点|缺点|
|-|-|
|双向励磁 → 变压器利用率高|VDS = 2×VIN (高压应力)|
|输出纹波频率 2×fSW|磁通不平衡风险 (伏秒积不对称)|
|双管驱动|变压器需中心抽头|
|适合低压大电流输入||

### 9.5 典型应用

* **低压输入隔离电源** (如 12V/24V→高压)
* **逆变器前级** (DC-AC)
* **通信电源** (48V→低压大电流)
* **UPS 逆变**

\---

## 十、Half-Bridge (半桥) 变换器

### 10.1 基本拓扑

```
       ┌── S1 ──┬── Cblock ──┬── Np ──┐
       │        │            │(隔直)    │
VIN ───┤  C1    │            └─────────┤
       │  ─     │                      │
       │  C2    ├──────────── Np ──────┤
       │        │                      │
       └── S2 ──┴────────────────────┘

C1, C2 = 输入分压电容 → 中点电压 = VIN/2
S1, S2 交替导通 (50% 占空比, 带死区)
```

### 10.2 工作原理

```
S1 ON: Np = +VIN/2  → 正向励磁
S2 ON: Np = −VIN/2  → 反向励磁

开关管电压应力:
VDS(max) = VIN(max)     (远优于 Push-Pull 的 2×VIN!)

变压器初级电压 = ±VIN/2
占空比控制 → 次级侧同 Forward/Push-Pull 输出整流+LC滤波
```

### 10.3 核心公式

```
变压器初级电压:
Vpri = ±VIN/2

开关管电压应力:
VDS(max) = VIN(max)          ← 关键优势

有效占空比 (每管):
Dmax < 0.5 (留死区)

输出:
VOUT = (VIN/2) × (Ns/Np) × (2×D)    (全波整流)
```

### 10.4 优缺点

|优点|缺点|
|-|-|
|VDS = VIN (vs Push-Pull 2×VIN)|需要两个输入分压电容|
|适合高压输入|变压器电压 = VIN/2 (利用率低)|
|双向励磁|Cblock 需处理全部初级电流|
|可做 LLC 谐振|占空比损失 (死区)|

### 10.5 典型应用

* **ATX 电脑电源主变换级** (AC-DC: 390V→12V)
* **通信电源** (48V→12V)
* **服务器电源前级**
* **UPS**
* **半桥 LLC 谐振** (见第13章)

\---

## 十一、Full-Bridge (全桥) 变换器

### 11.1 基本拓扑

```
       ┌── S1 ──┬──── Np ────┬── S3 ──┐
       │        │            │        │
VIN ───┤        │            │        ├── GND
       │        │            │        │
       └── S2 ──┴──── Np ────┴── S4 ──┘

对角开关导通:
  S1+S4 ON → Np = +VIN (正向)
  S2+S3 ON → Np = −VIN (反向)
```

### 11.2 工作原理

```
相移控制 (Phase Shift):
S1 ──┬──────┐        ┌──────
     │      │        │
     └──────┘        └──────┬──
S2 ──┐        ┌──────┐        ┌──
     │        │      │        │
     └────────┘      └────────┘
       dead    dead    dead
       time    time    time

S3, S4 与 S1, S2 移相 → 控制有效占空比
ZVS (零电压开关): 利用变压器漏感实现软开关
```

### 11.3 核心公式

```
变压器初级电压:
Vpri = ±VIN               (vs Half-Bridge 的 ±VIN/2 → 利用率翻倍)

开关管电压应力:
VDS(max) = VIN(max)       (与 Half-Bridge 相同)

有效占空比:
D\_eff = (相移角) / π

输出:
VOUT = VIN × (Ns/Np) × (2×D\_eff)      (全波整流)
```

### 11.4 优缺点

|优点|缺点|
|-|-|
|变压器电压 = ±VIN (最高利用率)|4 个 MOSFET (成本高)|
|VDS = VIN|驱动复杂 (需隔离驱动)|
|适合最大功率等级|可能出现直通|
|可实现 ZVS (移相全桥)||

### 11.5 典型应用

* **大功率 DC-DC** (1\~10kW+)
* **电动汽车 DC-DC** (400V→12V)
* **焊接电源**
* **电镀电源**
* **通信基站电源**
* **相移全桥 ZVS** (服务器电源 48V→12V)

## 十三、谐振变换器 (Resonant Converter)

### 13.1 为什么要谐振？

传统 PWM 变换器的开关管在电压和电流都不为零时开通/关断 → **硬开关**：

```
硬开关损耗:
P\_switching\_loss = 0.5 × VDS × ID × (tr + tf) × fSW
┌──────────────────────┐
│ fSW ↑ → P\_loss ∝ fSW │  → 高频化遇到瓶颈
└──────────────────────┘

软开关:
ZVS (Zero Voltage Switching): 电压先降到零 → 再开通 (消除 Coss 放电损耗)
ZCS (Zero Current Switching): 电流先降到零 → 再关断 (消除拖尾损耗)
```

### 13.2 LLC 谐振变换器

```
Half-Bridge LLC:

       ┌── S1 ──┬── Cr ── Lr ──┬── Np ──┐
       │        │    谐振槽      │        │
VIN ───┤  C1    │               │        │
       │  ─     │               │  Lm    │ ← 励磁电感 (参与谐振)
       │  C2    │               │        │
       └── S2 ──┴───────────────┴────────┘
       (半桥开关)

次级: 中心抽头全波整流 (或全桥整流)
```

**两个谐振频率**：

```
fr1 = 1 / (2π × √(Lr × Cr))
    → Lr 和 Cr 串联谐振 (Lm 被输出电压钳位, 不参与)

fr2 = 1 / (2π × √((Lr+Lm) × Cr))
    → Lr+Lm 与 Cr 谐振 (Lm 参与, 频率更低)
    → fr2 < fr1
```

**工作区域**：

```
         增益
          │
          │  Zone-1 (容性区, 禁止!)   Zone-2 (感性区, ZVS)
          │     ╲                  ╱
          │      ╲                ╱  ← 不同 Q 值曲线
    M=1 ──┼───────╲─────────────╱─────────
          │          ╲         ╱
          │           ╲───────╱
          │            ╲     ╱
          │             ╲   ╱
          └──────────────┼──┼──────────→ fSW
                        fr2 fr1

Q = √(Lr/Cr) / Rac  (品质因数)
Rac = 8/π² × Np²/Ns² × RLOAD  (等效交流负载)

ZVS 条件: 开关频率 > 谐振频率 (感性区)
  电流滞后电压 → 开关管开通前体二极管已导通 → VDS ≈ 0
```

**LLC 优缺点**：

|优点|缺点|
|-|-|
|全负载范围 ZVS (初级)|控制复杂 (变频控制)|
|次级 ZCS (关断)|轻载效率下降|
|效率极高 (95\~98%)|输入范围窄 (增益范围有限)|
|EMI 极小|启动/短路保护困难|
|频率可到 MHz 级|设计计算复杂|

**典型应用**：

* **大功率 AC-DC 适配器** (>100W)
* **服务器电源**
* **通信电源**
* **电动汽车 OBC (车载充电机，后级)**
* **LED 大功率驱动**
* **平板电视电源**

## 十四、同步整流 (Synchronous Rectification)

### 14.1 原理

```
传统二极管整流:              同步整流 (MOSFET):

  D                             S\_rect (MOSFET)
──▶├──                         ──┤  ├──
VF ≈ 0.5V (肖特基)              RDS(on) × I ≈ 0.1V @ 10A (RDS=10mΩ)


损耗对比 (IOUT=10A):
  二极管: PD = 10A × 0.5V = 5W
  MOSFET: PD = 10A² × 0.01Ω = 1W  → 省 4W!

适用: 低压大电流 (< 12V, > 2A)
不适用: 高压 (> 50V) → MOSFET RDS(on) 随耐压增大, 优势消失
```

### 14.2 驱动方式

|方式|原理|优点|缺点|
|-|-|-|-|
|**自驱**|变压器辅助绕组|简单|时序不精确|
|**控制 IC 驱动**|PWM IC 输出驱动|时序精确|IC 需支持|
|**智能整流 IC**|检测 VDS 导通/关断|最优|成本高|

## 十五、多相交错并联 (Multi-Phase Interleaving)

### 15.1 原理

```
Phase 1: S1 ── L1 ──┬──
Phase 2: S2 ── L2 ──┤
Phase 3: S3 ── L3 ──┼── VOUT
...                  │
Phase N: Sn ── Ln ──┘

各相 PWM 信号相位差 = 360°/N

效果:
  输入/输出纹波频率 = N × fSW
  纹波幅度 ↓ (各相纹波互相抵消)
  瞬态响应 ↑ (等效频率提升)
  每相功率 = Ptotal/N (热分布均匀)
```

### 15.2 应用

* **CPU/GPU 核心供电** (VRM: 12V→0.8\~1.2V, 100\~300A, 6\~12相)
* **大电流 POL**
* **48V→1V AI 加速器供电**

## 十六、选型决策树

```
确定需求参数:
  · VIN range, VOUT, POUT (或 IOUT)
  · 是否需要隔离？
  · EMI 要求？
  · 效率目标？
  · 尺寸/成本约束？

  │
  ├─ 不需要隔离？
  │    │
  │    ├─ VOUT < VIN (总是)?
  │    │    └──→ Buck
  │    │         ├─ IOUT < 10A → 单相 Buck
  │    │         ├─ IOUT 10\~50A → 单相 Buck + 同步整流
  │    │         └─ IOUT > 50A → 多相 Buck
  │    │
  │    ├─ VOUT > VIN (总是)?
  │    │    └──→ Boost
  │    │
  │    ├─ VOUT ⪋ VIN (输入跨过输出)?
  │    │    ├─ 可接受负压 → 反相 Buck-Boost
  │    │    ├─ 需要正压 → SEPIC 或 四开关 Buck-Boost
  │    │    └─ 宽范围 (>4:1) → SEPIC (输入电流连续)
  │    │
  │    └─ 超低噪声 (μV 级)?
  │         └──→ Buck/Boost + LDO (级联)
  │
  └─ 需要隔离？
       │
       ├─ POUT < 75W → Flyback
       │    ├─ 定频 → CCM Flyback
       │    └─ 变频/准谐振 → QR Flyback (谷底开关)
       │
       ├─ POUT 75\~300W → Forward (双管)
       │    ├─ 低压输入 → Push-Pull
       │    └─ 高压输入 → Two-Switch Forward
       │
       ├─ POUT 300\~1000W → Half-Bridge
       │    ├─ 高效率要求 → LLC 谐振半桥
       │    └─ 宽输入范围 → PWM Half-Bridge
       │
       └─ POUT > 1kW → Full-Bridge
            ├─ 最高效率 → LLC 全桥
            ├─ 宽范围 ZVS → 相移全桥 (PSFB)
            └─ > 5kW → 三相桥
```

## 十八、控制模式一览

|模式|原理|优点|缺点|
|-|-|-|-|
|**电压模式 (VMC)**|仅反馈 VOUT|简单，抗噪好|LC 双极点需 Type III 补偿|
|**峰值电流模式 (PCM)**|反馈 VOUT + IL 峰值|单极点 (简单补偿), 逐周期限流|需斜率补偿 (D>0.5), 对噪声敏感|
|**平均电流模式**|反馈平均 IL|精确电流控制|需电流误差放大器|
|**滞环 (Hysteretic)**|VOUT 直接控制开关|极快瞬态, 不需要补偿|变频, EMI 差|
|**恒定导通时间 (COT)**|固定 ton, 变 toff|瞬态快, 轻载效率高|变频, 需 ESR 产生纹波|
|**准谐振 (QR)**|谷底检测开通|ZVS, EMI 低|变频, 峰值电流高|
